# Fitness Trainer CRM Frontend

React приложение для управления CRM системой фитнес-тренера.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

## Структура

- `src/pages/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/services/` - API клиент
- `src/contexts/` - React контексты (Auth и т.д.)

## API

API должен быть запущен на `http://127.0.0.1:8009` (настроено в `vite.config.ts`)
